L.S.E. - Commandes

Une console peut être dans l'un des trois états suivants :

  1. État dormant : la console n'est pas en communication avec le système.
  2. État moniteur : le dialogue de l'utilisateur avec le système se fait à l'aide des commandes.
  3. État exécution : en mode exécution immédiate, la ligne d'instructioons est exécutée ; en mode exécution différée : le programme est exécuté.

Pour donner une commande au système, on tape les deux premières lettres de la commande, puis la combinaison de touches CTRL-S (X-OFF) pour la valider. Le système complète alors le libélé de la commande et attend éventuellement des paramètres supplémentaires (qui seront aussi validés par CTRL-S).

Liste des commandes

Connection & déconnection

BONJOUR
Permet de passer de l'état dormant à l'état moniteur.

AU REVOIR
Permet de passe de l'état moniteur à l'état dormant. Le programme, toutes ses données, et tous les fichiers temporaires sont effacés.

Commandes pour la programmation

Numéro_de_ligne Instruction...
Toute ligne commençant par un numéro est interprétée comme une ligne d'instruction à mémoriser dans le programme courant, à la place d'une éventuelle ligne préexistante de même numéro.

LISTER A PARTIR DE N1{,N2}
Affiche les lignes du programme, à partir de la ligne N1 jusqu'à la ligne N2 ou jusqu'à la fin. La touche ESC peut être utilisée pour arrêter le listage.

NUMERO A PARTIR DE N1{,N2}
Affiche les numéro des lignes du programme courant, à partir de la ligne N1 jusqu'à la ligne N2 ou jusqu'à la fin. La touche ESC peut être utilisée pour arrêter le listage.

EFFACER LIGNES N1{,N2{,N3...}}
Efface les lignes du programme dont les numéros sont donnés.
EFFACER LIGNES N1 A N2
Efface les lignes du programme dans l'intervale N1 à N2 inclus.
EFFACER LIGNES *
Efface tout le programme.

Commandes pour l'exécution et la mise au point

EXECUTER A PARTIR DE N1{,N2}
Passe dans l'état exécution. Exécute le programme en commençant par la ligne N1. L'exécution continue jusqu'à ce qu'on arrive à la ligne N2, ou à une instruction PAUSE ou TERMINER, ou qu'une erreur survienne, ou que l'on appuit sur la touche ESC.

PAS A PAS
Peut être utilisé avant EXECUTER, CONTInuer ou REPRENDRE, afin de faire arrêter l'exécution à chaque ligne de programme. L'exécution peut être reprise en tapant CTRL-S ou en donnant la commande CONTINUER.

NORMAL
Annule la commande PAS A PAS.

CONTINUER
Continue l'exécution du programme interrompu par la commande PAS A PAS, par une instruction PAUSE, ou par la toche ESC. L'exécution reprend à partir de l'endroit où elle fut interrompue.

POURSUIVRE JUSQU'EN N1
Continue l'exécution du programme comme CONTINUER, mais en indiquant un arrêt sur la ligne N1.

REPRENDRE A PARTIR DE N1{,N2}
Permet de reprendre l'exécution à partir d'une ligne différente de celle où elle fut interrompue. Les paramètres N1 et N2 ont la même signification qu'avec la commande EXECUTER.

Commandes relatives aux rubans perforés

ABREGER
Supprime l'affichage le libélé complet des commandes. (cf. IN EXTENSO).

IN EXTENSO
Annule la commande ABREGER.

SILENCE
Supprime l'affichage de tout ce que l'utilisateur tape au clavier. La touche ESC annule cette commande.

PERFORER A PARTIR DE N1{,N2}
Perfore les lignes du programme courant, à partir de la ligne N1 jusqu'à la ligne N2 ou jusqu'à la fin. La touche ESC peut être utilisée pour arrêter la perforation. Dans l'émulateur EMULSE, la perforation est virtuelle : on produit un ruban perforé virtuel rangé dans une armoire virtuelle. Le nom inscrit sur le ruban perforé virtuel est demandé à l'utilisateur avant la perforation.

RUBAN
Cette commande déclanche le lecteur de ruban perforés. Dans l'émulateur EMULSE, les rubans perforés sont virtuels. La commande donne la liste des rubans perforés virtuels rangés dans une armoire virtuelle. L'utilisateur saisi le nom inscrit sur le ruban perforé virtuel qu'il veut lire.

Commandes de gestion des fichiers

La plupart des commandes de gestion des fichiers sont soumises à l'utilisation préalable de la commande IDENTIFIER.

IDENTIFIER XXXXX
La clé secrète XXXXX permet à l'utilisateur de s'identifier auprès du système qui l'autorisera (ou non) à donner les commandes de gestion des fichiers suivantes.

TABLE DES FICHIERS
Donne la liste des fichiers appartement à l'utilisateur.

UTILISATION DU DISQUE
Donne la liste de tous les fichiers se trouvant sur le disque dur.

APPELER NOMPG
Cette commande charge un programme stocké sous le nom NOMPG.

LANCER NOMPG
Cette commande charge et lance l'exécution d'un programme stocké sous le nom NOMPG.

RANGER NOMPG
Cette commande enregistre le programme courant dans un fichier de nom NOMPG. Le fichier ne doit pas exister au préalable. (cf. MODIFIER NOMPG).

MODIFIER NOMPG
Cette commande enregistre le programme courant dans le fichier programme préexistant de nom NOMPG.

SUPPRIMER NOMFC,P
Supprime le fichier programme NOMFC.
SUPPRIMER NOMFC,D
Supprime le fichier données permanent NOMFC.
SUPPRIMER NOMFC,T
Supprime le fichier données temporaire NOMFC.
SUPPRIMER *
Supprime tous les fichiers données temporaires.

CATALOGUER NOMTE,NOMPE
Cette commande transforme le fichier temporaire NOMTE en fichier permanent de nom NOMPE.

DECODER NOMFC{,N1{,N2}
Enregistre dans un fichier donnée temporaire de nom NOMFC le source du programme courant, de la ligne N1 à la ligne N2 incluses.

ENCODER NOMFC{,N1{,N2}
Charge les lignes sources présentes dans le fichier donnée temporaire de nom NOMFC de la ligne N1 à la ligne N2 incluses.
