Déclarations

Les variables scalaires (nombres) n'ont pas besoin d'être déclarées (mais elles doivent être initialisées).

Un identificateur est composé d'une lettre, suivit d'au plus 4 caractères alphanumériques (lettre ou chiffre).

CHAINE identificateur{,identificateur...}
Déclare les identificateurs comme étant des chaines de caractères.

TABLEAU identificateur[largeur{,hauteur}]{,identificateur[largeur{,hauteur}]...}
Déclare les identificateurs comme étant des tableaux de nombre de telle largeur et de telle hauteur.

LIBERER identificateur{,identificateur...}
Supprime les variables dénotées par les identificateurs. Ces identificateurs peuvent être réutilisés dans une nouvelle déclaration, ou comme variable scalaire.

identificateur _ expression
Affecte à la variable identificateur la valeur de l'expression. (Le caractère _ s'affichait comme une flêche vers la gauche (<-) sur les consoles de l'époque).
